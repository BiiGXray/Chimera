# Chimera, Gaggle, DEL and Data Augmentation Sensitivity Landscapes
The Chimera and Gaggle Algorithms, as well as the DEL generator and Data Augmentation sensitivity landscape analysis code corresponding to the paper "On the Fragility of Neural Architecture Search: The Role of Overfitting and Task Complexity in Medical Image Analysis" to be published in Frontiers for Artificial Intelligence is provided herein.

The data to replicate the experiments on the CT misalignment dataset will be provided shortly.
